# Individual Practice

Individual Practice sessions focus on one player. They include access to the Mechanics Assessment logger in addition to the standard four loggers.

## When to Use Individual Practice

- One-on-one skill sessions between coach and player
- Pre-game warm-up pitching evaluation
- Post-game review of technique
- Mechanics assessments (required for mechanics analytics)

## Starting an Individual Practice Session

1. From the dashboard tap **Log Practice Data**
2. Select **Individual Practice**
3. Select the player from your roster
4. Tap **Start Session**

All loggers now default to the selected player. You can still change the player in any logger — useful if you assess multiple players back to back in the same session.

## Mechanics Assessments in Individual Practice

The **Mechanics Assessment** option appears in the Practice Hub when in Individual Practice mode. Tap it to open the assessment logger for the selected player's sport and position.

See [Mechanics Assessments](/practice/mechanics-assessments) for complete assessment instructions.

## Post-Session Notes

At the end of an Individual Practice session the app prompts you to add session notes. These notes connect to the player's Coach Notes history. Use this prompt to capture observations while they are fresh — it takes less than a minute and creates a permanent record of what was observed and worked on.
