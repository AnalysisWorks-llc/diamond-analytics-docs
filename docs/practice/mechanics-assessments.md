# Mechanics Assessments

Mechanics assessments are conducted during Individual Practice sessions. They capture technique scores that appear in the Mechanics Analytics section of each player's profile.

## How to Conduct an Assessment

1. Start an Individual Practice session and select the player
2. From the Practice Hub tap **Mechanics Assessment**
3. Select the mechanics area to assess: Hitting, Pitching, Fielding, or Baserunning
4. Work through each element, scoring 0–3
5. Add optional notes for any element
6. Tap **Save Assessment** when complete

You can assess multiple areas in the same session — return to the Practice Hub between each assessment and start a new one.

## Scoring Each Element

Use the 0–3 scale for each element:

| Score | Meaning |
|-------|---------|
| 0 | Not observed |
| 1 | Needs significant work |
| 2 | Developing |
| 3 | Correct |

**0 (Not observed)** is for situations where you could not evaluate the element — player was wearing catching gear, the drill did not include that phase, etc. A 0 does not count against the player's average for that element.

## Assessment Notes

Each element has a notes field. Use it to document what specifically you observed — this note appears in the player's mechanics history and can be referenced when writing report card comments.

Good note: "Hip rotation initiates before hands — timing issue, not flexibility"
Poor note: "Needs work"

## Viewing Assessment Results

Assessment results appear immediately in the player's analytics under **Mechanics**. The radar chart updates with each new assessment. After 3 or more assessments trend charts become available showing improvement over time.

## Softball vs Baseball

Pitching mechanics assessments use sport-specific elements. Softball assessments cover the windmill delivery (arm circle, stride, hip drive, release, follow-through). Baseball assessments cover the traditional delivery (balance, stride, arm action, rotation, finish). Select the correct sport when creating your team — assessments adjust automatically.
