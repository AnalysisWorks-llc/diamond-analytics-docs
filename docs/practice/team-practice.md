# Team Practice

Team practice sessions track data across your full roster during a practice. All four loggers are available — pitching, at-bat, fielding, and baserunning.

## Common Team Practice Scenarios

### Bullpen Session

Log every pitch thrown during bullpen work using the Pitching Logger. Select the pitcher at the top of the screen and log each pitch with type, location, and result. Bullpen data counts toward pitching analytics and helps track command development between games.

### Batting Practice

Log at-bats during live BP rounds using the At-Bat Logger and Pitching Logger together. This gives you spray chart and contact type data on every hitter without needing a live game.

### Fielding Drills

Log fielding repetitions using the Fielding Logger. This is particularly useful for tracking which players need work on specific ball types (ground balls, fly balls) or directional zones.

### Voice Logging at Practice

Voice logging is especially efficient at practice where repetitions happen faster than manual entry allows. A parent volunteer with the Voice Logger can track a full bullpen session or batting practice round without slowing down the drill.

## Multiple Loggers at Once

During team practice multiple coaches or parents can be logging simultaneously from different devices. All data syncs to the same session in real time.

For example:
- One coach logs pitching from behind the mound
- A parent volunteer logs hitting from behind the backstop using voice logging
- Both streams appear in the same session analytics
