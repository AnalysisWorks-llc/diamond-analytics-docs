# Desktop Voice Logging

Desktop voice logging uses the browser's built-in Web Speech API. It works without any keyboard setup.

## How to Use

1. Open the Voice Logger from the Game Hub or Practice Hub
2. Click the **microphone button**
3. Speak your play description clearly
4. The recording stops automatically after a brief pause
5. Review the confirmation card and tap **Confirm and Save**

## Tips for Desktop

**Speak at a natural pace** — the browser's speech recognition performs better with normal speaking speed than slow, over-enunciated speech.

**Pause between plays** — the auto-stop detects silence. After you finish speaking wait a moment for the recording to stop and the AI to process.

**Timeout behavior** — if you pause too long mid-description the recording may stop early. If this happens your partial description may not parse correctly. Tap **Try Again** and speak the complete play description without long pauses.

**Background noise** — desktop microphones pick up more ambient noise than phone microphones held closer to the mouth. Work in a reasonably quiet environment or use a headset microphone.

## Browser Microphone Permission

The first time you use voice logging your browser will request microphone access. Click **Allow**. If you accidentally denied it:

- **Chrome:** Click the lock icon in the address bar → Site settings → Microphone → Allow
- **Edge:** Same process via the lock icon
- **Safari (Mac):** System Preferences → Security & Privacy → Microphone → check the browser
