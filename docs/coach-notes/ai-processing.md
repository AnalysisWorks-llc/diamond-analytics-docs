# AI Note Processing

AI note processing converts free-form coaching observations into structured player-specific records. It is designed for coaches who prefer to write or dictate extended notes rather than entering observations field by field.

## How to Use

1. Navigate to **Notes hub → Process Notes with AI**
2. Type or paste your observations (up to 2,000 characters)
3. Select a date range for context (optional but recommended)
4. Tap **Process Notes with AI**
5. Review results on the structured output screen

## What the AI Extracts

The AI identifies:
- Which player each observation is about (matched to your roster)
- What area of the game the observation covers (pitching, hitting, fielding, baserunning, mechanics)
- Whether the observation is positive, a concern, or neutral
- Whether the issue is recurring (based on previous notes in the context window)

## Results Screen

The results screen shows a card for each extracted observation:
- **Player** — matched roster player (amber warning if not matched)
- **Area** — hitting, pitching, fielding, baserunning, or mechanics
- **Observation** — the structured summary
- **Pattern** — Recurring, Improving, or New (based on note history)

From each card you can:
- **Save** — saves as a player note
- **Create Training Plan** — opens a pre-filled training plan
- **Skip** — discards this observation

## Including Previous Notes as Context

When you select a date range the AI includes notes from that period as background context. This enables pattern detection — the same issue noted three times becomes **Recurring** instead of **New**.

Use 2-4 weeks as the standard context window. A full season can work but may surface too many resolved issues.

## Usage Limits

AI note processing counts toward your monthly AI note limit:

| Plan | AI Notes per Month |
|------|--------------------|
| Coach | 50 |
| Organization | 200 |

Each **Process Notes with AI** submission counts as one use regardless of how many player observations it extracts.
