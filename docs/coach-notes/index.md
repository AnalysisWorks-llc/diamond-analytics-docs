# Coach Notes

## Overview

Coach Notes captures coaching observations and connects them to training plans. Notes can be entered as text, spoken as voice notes, or processed through AI to extract structured player observations from free-form text.

## Adding Notes

Notes can be added from multiple places:
- **Quick Note button** on the dashboard
- **Notes tab** on any player profile
- **Notes hub** via the bottom navigation More menu
- **Voice Note** button for spoken observations
- **End of practice** prompt in Individual Practice sessions

## Note Types

| Type | Description |
|------|-------------|
| Team | General team observation |
| Player | Observation about a specific player |
| Game | Note tied to a specific game |
| Practice | Note tied to a practice session |

## AI Note Processing

The AI processing feature extracts structured player observations from free-form notes. This is useful when you have written or spoken extended observations across multiple players.

**How to use it:**
1. Navigate to Notes hub → **Process Notes with AI**
2. Type or paste your notes (up to 2,000 characters)
3. Select the date range for context — previous notes from that period are included to help identify patterns
4. Tap **Process Notes with AI**
5. Review the extracted observations on the results screen
6. Save individual observations or create training plans directly

## Pattern Recognition

When you include previous notes as context the AI identifies recurring issues:
- Issues noted multiple times are flagged as **Recurring**
- Issues that appear resolved are flagged as **Improving**
- The overall pattern summary appears at the top of results

## Voice Notes

Tap **Voice Note** to speak your observations directly. The app captures your speech and sends it through the same AI processing pipeline. Ideal for coaches who want to dictate observations right after a game or practice.

**On mobile:** The keyboard appears with the microphone icon ready. Tap the keyboard microphone, speak your observations freely, then tap **Process Notes with AI** when done.
