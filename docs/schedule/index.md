# Schedule and Tournaments

## Overview

The Schedule screen tracks your upcoming and completed games and tournaments. Pre-scheduling games connects them to game sessions so you avoid duplicate data entry.

## Scheduling a Game

1. Navigate to **Schedule**
2. Tap **+ Game**
3. Select opponent from your saved opponents list or add a new one
4. Enter date, time, and location
5. Select Home or Away
6. Save

## Starting a Scheduled Game

When you start a Game session the app checks for scheduled games and presents them as options. Selecting a scheduled game pre-fills all game details automatically.

## Tournaments

Travel teams typically play 4-8 games over a tournament weekend with pool play followed by bracket play.

**Adding a tournament:**
1. Tap **+ Tournament**
2. Enter tournament name, location, dates, and format
3. Save

**Adding games to a tournament:**
From the tournament detail screen add pool play games (opponents known in advance) and bracket games (opponents TBD until bracket is announced).

**TBD opponents:**
Schedule bracket game slots without knowing the opponent. When the bracket is announced edit the game to add the opponent name.

**Tournament results:**
Mark each game result as it is played. When the tournament is complete tap **Mark Complete** and record your final placement — Champion, Runner-Up, Semi-Final, Quarter-Final, or Pool Play Only.

Champion and Runner-Up placements display as badges on the tournament card and contribute to your season summary on the dashboard.

## Recording Game Results

After a game tap **Enter Result** on the game card. Enter the final score — Win/Loss/Tie calculates automatically.

If the game was scored live through the Game Hub the result saves automatically when you end the game session.
