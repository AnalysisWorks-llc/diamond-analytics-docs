# Scheduling Games

Pre-scheduling games saves time on game day and keeps your season record organized.

## Adding a Game

1. Navigate to **Schedule**
2. Tap **+ Game**
3. Fill in the details:
   - **Opponent** — select from saved opponents or tap **New Opponent** to add
   - **Date** — tap to open the date picker
   - **Time** — optional but useful for the daily schedule view
   - **Location** — field name or address
   - **Home or Away** — affects how the score is displayed
4. Tap **Save**

The game appears on your schedule and in the dashboard upcoming games widget.

## Managing Opponents

Opponents are saved automatically the first time you add them. On subsequent games against the same team select them from your opponent list — this keeps game history organized by opponent.

Edit or delete saved opponents from **Schedule → Opponents** in the menu.

## Game Day Connection

On game day when you start a game session the app checks for games scheduled for today. Tap the scheduled game to pre-fill the opponent and game details. This links the scoring session to the scheduled game so the result records automatically when you end the session.

## Editing a Scheduled Game

Tap any game card on the schedule to open game details. Tap **Edit** to modify date, time, opponent, or location. You can edit scheduled games up until the game session is started.

## Cancellations and Postponements

To mark a game as cancelled or postponed open the game detail and tap **Mark as Cancelled** or **Edit** to change the date. Cancelled games remain on the schedule with a Cancelled badge and do not count toward your season record.
