# Baserunning Logger

The Baserunning Logger tracks stolen base attempts, pickoffs, and scoring events. This data drives stolen base percentage, caught stealing rate, and baserunning analytics.

## Logging a Baserunning Event

1. Select the **runner** from the player selector
2. Select the **base** the runner is attempting to advance to or was on
3. Select the **result**
4. The record saves automatically

## Bases

| Selection | Description |
|-----------|-------------|
| 1B | Runner on first base / advancing to first |
| 2B | Runner attempting to steal second or advancing to second |
| 3B | Runner attempting to steal third or advancing to third |
| Home | Runner scoring |

## Results

| Result | Description |
|--------|-------------|
| Stolen Base | Successful steal |
| Caught Stealing | Runner thrown out attempting to steal |
| Picked Off | Runner picked off a base |
| Scored | Runner scored a run |

## Quick Navigation

The arrow button in the top right navigates directly to the **At-Bat Logger** — useful for quickly switching between logging the batter and logging baserunning events for runners already on base.
