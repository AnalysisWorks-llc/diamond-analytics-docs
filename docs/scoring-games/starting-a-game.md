# Starting a Game Session

## Before the First Pitch

1. From the dashboard tap **Log Game Data**
2. Select **Game** as the session type
3. Select your opponent — choose from your saved opponents list or tap **New Opponent** to add one
4. If you have a game scheduled for today it will appear at the top as a pre-filled option — select it to avoid re-entering details
5. Confirm the date and location
6. Tap **Start Game**

The Game Hub opens and the session is active. An amber active game banner will appear on all other screens until the game is ended.

## Pre-Game Checklist

Before the first pitch confirm:

- Correct opponent is selected
- Correct date shows (important for retroactive scoring)
- Your roster is up to date — add any new players before starting
- Any scorekeeper helpers are logged in and have the app open

## Scrimmage vs. Game

When starting a session you select the session type:

- **Game** — counts toward season records, win/loss, ERA, and all analytics
- **Scrimmage** — logged identically but filtered separately in analytics

Use Scrimmage for intra-squad games or unofficial opponents. The event type filter in analytics lets you compare game vs scrimmage performance.

## Resuming an Interrupted Session

If you navigate away mid-game the session stays active. Return to it via:

- The **amber active game banner** at the top of any screen
- The **Resume** button on the game card in your schedule
- The **Live Game widget** on your dashboard
