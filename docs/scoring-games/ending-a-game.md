# Ending a Game

## End the Game Session

When the game is over:

1. From the Game Hub tap **End Game**
2. A confirmation screen appears asking you to verify the final score
3. Confirm the result — Win, Loss, or Tie is calculated automatically from the scores
4. Tap **Confirm** to end the session

## What Happens When You End a Game

- The game session status changes from Active to Complete
- The final score and result are saved to your schedule
- Your season record on the dashboard updates
- The active game banner disappears from all screens
- All logged stats remain associated with this game

## Resuming an Interrupted Game

If you navigate away from the game mid-session the session remains active. You can return to scoring from:

- The **amber active game banner** that appears at the top of every screen
- The **Resume button** on the active game card in your schedule
- The **Live Game widget** at the top of your dashboard

The game session does not end until you explicitly tap End Game.

## Retroactive Scoring

If you could not score a game live you can enter stats later:

1. From the Game Hub tap **Start New Session**
2. Select **Unscored Past Games** — any scheduled game without stats appears here
3. Select the game and log stats from video or memory
4. End the session normally when complete
