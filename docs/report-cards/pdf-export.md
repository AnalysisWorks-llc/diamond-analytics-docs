# PDF Export

## Exporting to PDF

1. Generate the report card (see [Generating Report Cards](/report-cards/generating))
2. In the preview tap **Export PDF**
3. The PDF generates and downloads to your device
4. Share or print from your device's PDF viewer

## PDF vs Share Link

| | Share Link | PDF Export |
|--|------------|------------|
| Requires internet | Yes (to view) | No (after download) |
| Printable | Browser print | Yes, optimized |
| Charts rendered | Yes | Yes |
| Expiration | 30 days | Never |
| File to manage | No | Yes |

Use **Share Link** for digital delivery — it always reflects the latest data if regenerated.
Use **PDF Export** for printing, attaching to email, or offline reference.

## Printing Tips

The PDF is formatted for standard letter size (8.5 x 11 inches). Print settings:
- Page size: Letter
- Orientation: Portrait
- Margins: Default (the PDF includes its own margins)
- Scale: 100% (do not fit to page — it is already sized correctly)

## PDF Contents

The PDF includes everything shown in the preview — stats, charts, mechanics radar charts, training plan summary, and coach comments. Charts are rendered as vector graphics and print cleanly at any resolution.
